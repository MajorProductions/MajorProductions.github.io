Random Number Generator
(Console Application)

This console application was developed for the sole reason of creating a functioning, lightweight, and fast solution for creating a set of unique randomized numbers. Currently, the Console Application weighs in at only 12kb and runs incredibly smoothly. For the remainder of the project, I will be working on making it smoother, prettier, and over-all a better experience.

Nerd Application

Going by "Nerd App" for short my Nerd Application will allow the user to complete various activities at the touch of a hand. Being mainly focused on functionality rather than design... the current UI is not as pleasant as it could be, but I plan on working on the design over time. The current feature(s) included in Nerd Application is Random Number Generation in the formats of Unique, Pseudo, and True random. The output from the generator can be formatted to the users liking on the fly. The current goal is to improve the functionality of the Random Generator making it more lightweight and cutting down on the time required to generate the numbers.

Major Solution
(Unity Library)

Major Solution is a multi-functional C# library intended for use with the Unity game creation software. The current version of Major Solution allows for quick access to Audio Sources and simple Math features that were not included in the base project. However, I feel a lot of work still needs to go into Major Solution before I feel ready to offer it to others.
